# java-cli-buildr-dolt-simple

## Description
A java buildr build, that connects to dolt database.

## Tech stack
- java
- buildr
  - 6.8.2

## Docker stack
- dolthub/dolt-sql-serverdb:latest
- vanto/apache-buildr:latest-jruby-jdk8

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
